# Threejs-template
threejs template 
threejs模板(开发学习用)

### 1.**包含**

**1.场景**

**2.摄像机**

**3.轨道控制器(已经开启阻尼)**

**4.一个平面**

**5.resize**

**6.时钟**

### 2.使用方法

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
