# Threejs-template
threejs template 
threejs模板(开发学习用)

### 1.**包含**

**1.场景**

**2.摄像机**

**3.轨道控制器(已经开启阻尼)**

**4.一个平面(基础网格材质 和 着色器材质)**

**5.resize**

**6.时钟**

**7.三种噪波贴图**

### 2.使用方法

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
